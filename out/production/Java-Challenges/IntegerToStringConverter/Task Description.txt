BG
Описание на задачата:

Задачата изисква да вземеш цяло число като вход и да го конвертираш в низ (стринг). След това трябва да провериш дали конвертирането е успешно. Ако успешно си извършил конверсията, програмата трябва да отпечата "Good job". Ако възникне грешка при процеса на конвертиране, тя трябва да отпечата "Wrong answer".

Основни стъпки:
Прочиташ цяло число от входа.
Конвертираш това число в низ.
Проверяваш дали конвертираното число може да бъде правилно интерпретирано обратно като цяло число./В зависимост от резултата, програмата ще отпечата съответното съобщение ("Good job" или "Wrong answer").

Вход:
100

Изход:
Good job

EN
Task description:

The task requires you to take an integer as input and convert it into a string. Then, you need to check if the conversion is successful. If the conversion is successful, the program should print "Good job". If an error occurs during the conversion process, it should print "Wrong answer".

Main steps:

Read an integer from input.
Convert the integer into a string.
Check if the converted string can be correctly interpreted back as an integer.
Depending on the result, the program will print the appropriate message ("Good job" or "Wrong answer").

Input:
100

Output:
Good job
